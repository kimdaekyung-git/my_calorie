# Design System (기초 디자인 시스템) - 헬스메이트

> 깔끔하고 차분한, 담백하고 간결한 디자인을 지향합니다.
> 헤드스페이스의 미니멀함을 참고합니다.

---

## MVP 캡슐

| # | 항목 | 내용 |
|---|------|------|
| 1 | 목표 | 귀찮은 입력 과정을 간소화하여 식단을 꾸준히 기록할 수 있게 하기 |
| 2 | 페르소나 | 다이어트 중인 사람 |
| 3 | 핵심 기능 | FEAT-1: 간편 식단 기록 |
| 4 | 성공 지표 (노스스타) | 매일 3끼 기록 달성률 |
| 5 | 입력 지표 | 일일 기록 횟수, 주간 연속 기록 일수 |
| 6 | 비기능 요구 | 식단 입력 3초 이내 완료 |
| 7 | Out-of-scope | 운동 기록, 소셜, 다크 모드, 회원가입/로그인 |
| 8 | Top 리스크 | 입력 귀찮아서 3일 만에 포기 |
| 9 | 완화/실험 | 음식 검색 자동완성 + 최근 기록 재사용 |
| 10 | 다음 단계 | 식단 기록 화면 프로토타입 구현 |

## 1. 디자인 철학

### 1.1 핵심 가치

| 가치 | 설명 | 구현 방법 |
|------|------|----------|
| 단순함 | 필요한 정보만 보여준다 | 화면당 하나의 주요 액션, 불필요한 장식 제거 |
| 차분함 | 부드럽고 편안한 느낌 | 파스텔 톤 색상, 넉넉한 여백, 부드러운 모서리 |
| 명확함 | 수치와 정보를 한눈에 | 큰 숫자, 명확한 계층 구조, 담백한 톤 |

### 1.2 참고 서비스 (무드보드)

| 서비스 | 참고할 점 | 참고하지 않을 점 |
|--------|----------|-----------------|
| 헤드스페이스 | 차분한 컬러, 넉넉한 여백, 부드러운 인터랙션 | 과한 일러스트, 캐릭터 |
| Apple 건강 | 깔끔한 수치 표현, 카드 레이아웃 | 복잡한 차트, 다양한 메뉴 |

## 5. 기본 컴포넌트

### 5.1 버튼 (Button)

| 상태 | Primary | Secondary | Ghost |
|------|---------|-----------|-------|
| 기본 | Sage Green 채움 + 흰 텍스트 | 테두리 + Sage 텍스트 | 텍스트만 |
| 호버 | 약간 어둡게 (#5A7D6E) | 배경 연한 채움 | 밑줄 |
| 포커스 | Sage 포커스 링 | Sage 포커스 링 | Sage 포커스 링 |
| 비활성 | 50% 투명도 | 50% 투명도 | 50% 투명도 |
| 로딩 | 스피너 표시 | 스피너 표시 | 스피너 표시 |

**크기:**
- Large: 높이 48px, 패딩 24px (주요 CTA)
- Medium: 높이 40px, 패딩 16px (기본)
- Small: 높이 32px, 패딩 12px (인라인)

### 5.2 입력 필드 (Input)

| 상태 | 스타일 |
|------|--------|
| 기본 | Border 컬러 테두리 + Surface 배경 |
| 포커스 | Primary 컬러 테두리 + 포커스 링 |
| 에러 | Error 컬러 테두리 + 에러 메시지 |
| 비활성 | 회색 배경, 입력 불가 |

### 5.3 카드 (Card) - 식단 기록 카드

- 배경: Surface (`#F8F6F2`)
- 테두리: 1px solid Border (`#E8E4DF`)
- 모서리: 12px 라운드
- 그림자: 0 1px 3px rgba(0,0,0,0.05)
- 내부 여백: md (16px)
- 구조: 음식 이름 (H3) + 칼로리 (Secondary 컬러) + 시간 (Caption)

### 5.4 FAB (Floating Action Button) - 기록 추가

- 위치: 화면 하단 우측
- 크기: 56x56px
- 색상: Primary (Sage Green)
- 아이콘: + (흰색)
- 그림자: 0 4px 12px rgba(107, 144, 128, 0.3)
- 모서리: 완전 원형 (28px)

### 5.5 끼니 탭 (Meal Type Tabs)

- 스타일: 수평 탭 (아침 / 점심 / 저녁 / 간식)
- 활성: Primary 텍스트 + 하단 2px 인디케이터
- 비활성: Text Secondary
- 높이: 44px

## 6. 접근성 체크리스트

### 6.1 필수 (MVP)

- [ ] **색상 대비**: 텍스트와 배경의 대비율 4.5:1 이상
- [ ] **포커스 링**: 키보드 탐색 시 포커스 표시 명확
- [ ] **클릭 영역**: 버튼/링크 최소 44x44px (모바일 터치 고려)
- [ ] **에러 표시**: 색상만으로 구분하지 않음 (아이콘/텍스트 병행)
- [ ] **폰트 크기**: 본문 최소 14px

### 6.2 권장 (v2)

- [ ] 키보드 전체 탐색 가능
- [ ] 스크린 리더 호환 (ARIA 라벨)
- [ ] 애니메이션 줄이기 옵션

## 7. 아이콘 & 일러스트

### 7.1 아이콘 라이브러리

| 옵션 | 설명 |
|------|------|
| Lucide | 깔끔한 라인 아이콘, 미니멀 스타일에 적합 |

### 7.2 아이콘 사용 규칙

- 크기: 16px (작음), 20px (기본), 24px (큼)
- 색상: 텍스트와 동일하게 상속
- 버튼 내: 텍스트 왼쪽에 배치, 8px 간격
- 끼니 아이콘: 아침(해), 점심(태양), 저녁(달), 간식(쿠키)
